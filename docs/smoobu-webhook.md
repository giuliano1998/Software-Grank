# Webhook de Smoobu

## Campos esperados
- id
- firstname
- lastname
- channelName
- apartmentId
- arrival
- departure
- adults
- children
- email
- phone

## Ejemplo de payload
```json
{
  "id": 119719486,
  "firstname": "Ada",
  "lastname": "Lovelace",
  "channelName": "Demo Channel",
  "apartmentId": 2,
  "arrival": "2024-07-01",
  "departure": "2024-07-05",
  "adults": 2,
  "children": 1,
  "email": "ada@example.com",
  "phone": "+34123456789"
}
```
